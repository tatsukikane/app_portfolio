# アプリポートフォリオ

あなたの毎日をもっと楽しくするアプリを紹介するポートフォリオサイトです。

## 🌐 デプロイURL

[https://app-portfolio-rho.vercel.app/](https://app-portfolio-rho.vercel.app/)

## 🚀 セットアップ

### 必要な環境

- Node.js 18以上
- npm / yarn / pnpm / bun

### インストール

```bash
npm install
# または
yarn install
# または
pnpm install
# または
bun install
```

### 開発サーバーの起動

```bash
npm run dev
# または
yarn dev
# または
pnpm dev
# または
bun dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

### ビルド

```bash
npm run build
npm start
```
